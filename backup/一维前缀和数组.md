# 前缀和的作用
快速求出数组中，某一段的区间和
# 创建前缀和数组F
F[i]表示：区间 **[1, i]** 中，所有元素的和
设原数组为 **a** ，根据定义 **F[i] = f[i - 1] + a[i]**，预处理出来一个前缀和数组
# 利用前缀和数组
如果要求原数组 **a** ,在区间 **[L, R]** 上的和，那么利用前缀和数组的性质：F[L - 1] 表示区间 **[1, L - 1]** 的和，F[R] 表示区间 **[1, R]** 的和
那么区间 **[L, R]** 的和就等于 **F[R] - F[L - 1]** 
# 题目
[一维前缀和](https://ac.nowcoder.com/acm/problem/226282)
---

对于给定的长度为 nn 的数组 {a1,a2,…,an}{a1​,a2​,…,an​} ，我们有 mm 次查询操作，每一次操作给出两个参数 l,r ，你需要输出数组中第 l 到第 r 个元素之和。

---
```c++
#include<iostream>
using namespace std;
typedef long long LL;
const int N = 1e5 + 10;
LL a[N];//原数组
LL f[N];//前缀和数组
int main()
{
    int n, m;
    cin >> n >> m;
        for(int i = 1; i <= n; i++)
        {
            cin >> a[i];
            f[i] = f[i - 1] + a[i];//预处理前缀和数组
        }
    while(m--)
    {
        int l, r;
        cin >> l >> r;
        cout << f[r] - f[l - 1] << endl;//前缀和数组的使用
    }
    return 0;
}
```
 *注意事项*：使用前缀和数组时，下标必须从1开始计数，` f[i] = f[i - 1] + a[i];//预处理前缀和数组`如果下标为0开始计数，在预处理时i等于0时会越界访问
