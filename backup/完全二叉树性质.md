# 做题时完全二叉树的一些性质
1. 完全二叉树最多只有一个度为1的结点，**即n<sub>1</sub>=0或1** 
2. 由n<sub>0</sub>+n<sub>1</sub>+n<sub>2</sub>=n<sub>1</sub>+2n<sub>2</sub>+1(结点数等于所有度的数量+1)可以得到n<sub>0</sub>=n<sub>2</sub>+1。又由n<sub>0</sub>=n<sub>2</sub>+1得到n<sub>0</sub>+n<sub>2</sub>=2n<sub>2</sub>+1，得到重要性质**n<sub>0</sub>+n<sub>2</sub>一定是奇数**
3. **只要知道完全二叉树的节点数，就可以推出度为0,1,2（n<sub>0</sub>，n<sub>1</sub>，n<sub>2</sub>）的结点个数**  
4. 若完全二叉树有2k（偶数）个结点  
由条件：
* n<sub>0</sub>+n<sub>2</sub>是奇数
* n<sub>0</sub>+n<sub>1</sub>+n<sub>2</sub>=2k  
**得到n<sub>1</sub>肯定是1，所以n<sub>1</sub>=1,n<sub>0</sub>=k,n<sub>2</sub>=k-1**
5. 若完全二叉树有2k-1（奇数）个结点 **n<sub>1</sub>=0,n<sub>0</sub>=k,n<sub>2</sub>=k-1**
